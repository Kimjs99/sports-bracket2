# 🏀 학교스포츠클럽 농구 토너먼트 대진 관리 시스템

학교스포츠클럽 농구 대회의 대진 생성, 경기 일정 관리, 결과 입력, 순위 확인을 위한 웹 애플리케이션입니다.  
중등부·고등부를 각각 독립적으로 관리할 수 있으며, 관리자와 일반 관람자(게스트) 역할을 구분합니다.

---

## 사용 가이드

### 관리자 (Admin)

대진 생성, 일정 입력, 경기 결과 입력, 공지 작성 등 모든 편집 기능은 **관리자 로그인** 후 사용할 수 있습니다.

#### 1. 관리자 계정 만들기

앱을 처음 실행하면 관리자 계정이 없는 상태입니다.

1. 화면 우하단 **자물쇠(🔒) 버튼** 또는 헤더의 **관리자** 버튼을 클릭합니다.
2. 처음 실행 시 **계정 만들기** 화면이 표시됩니다.
3. 사용할 **아이디**와 **비밀번호**(4자 이상)를 입력하고 확인 후 저장합니다.
4. 이후 방문 시에는 동일 계정으로 **로그인**합니다.

> 계정 정보는 브라우저 로컬 스토리지에 저장됩니다. 브라우저를 바꾸거나 데이터를 초기화하면 재설정이 필요합니다.

---

#### 2. 새 대회 만들기 (대진 생성)

1. 홈 화면에서 **고등부** 또는 **중등부** 탭을 선택합니다.
2. 해당 부에 대회가 없으면 **새 대진 만들기** 버튼이 표시됩니다.
3. 대회가 이미 있다면 오른쪽 상단 **새 대진** 버튼(점선 테두리)을 클릭합니다.
4. **대진 설정 화면**에서 참가 팀을 입력합니다.
   - 팀명을 입력하고 **추가** 버튼 또는 `Enter` 키로 추가합니다.
   - 이미 추가된 팀은 클릭하여 이름을 수정하거나 **삭제**할 수 있습니다.
   - 최소 2팀, 최대 64팀까지 등록 가능합니다.
5. **대진 생성** 버튼을 클릭하면 자동으로 토너먼트 브라켓이 생성됩니다.
   - 팀 수에 맞춰 2의 거듭제곱 브라켓이 생성되며, 부족한 슬롯은 **부전승**으로 채워집니다.

---

#### 3. 대진 재구성 (리셔플)

대진표를 새로 뽑고 싶을 때 사용합니다.

1. **대진표** 화면 → **재구성** 버튼 클릭 (자물쇠 아이콘이 있으면 로그인 필요)
2. 확인 팝업에서 **재구성** 클릭 시 팀 배정이 새로 무작위 생성됩니다.

---

#### 4. 경기 일정 입력

1. **대진 관리** 버튼(홈 대시보드 또는 대진표 화면)으로 이동합니다.
2. 상단 탭에서 **일정 입력**을 선택합니다.
3. 각 경기 카드에서 **날짜, 시간, 장소**를 입력하고 저장합니다.
4. 입력된 일정은 대진표 카드와 학교별 조회 화면에서 확인할 수 있습니다.

---

#### 5. 경기 결과 입력

1. 홈 대시보드에서 **경기 진행** 버튼을 클릭합니다. (관리자 전용 페이지)
2. 라운드 탭(예: 8강, 준결승)을 선택합니다.
3. 해당 경기 카드에서 **홈팀과 어웨이팀의 점수**를 입력하고 **저장**합니다.
   - 동점 입력 시 오류 메시지가 표시됩니다. (연장전 결과를 입력해야 합니다)
4. 저장 후 승자가 다음 라운드에 자동으로 반영됩니다.
5. 잘못 입력한 경우 **수정(연필 아이콘)** 버튼으로 결과를 재입력할 수 있습니다.

---

#### 6. 공지 작성 및 삭제

1. 홈 대시보드 또는 대시보드 화면의 **공지** 탭을 선택합니다.
2. 관리자 로그인 상태에서 **제목과 내용**을 입력 후 **공지 등록**을 클릭합니다.
3. 등록된 공지 오른쪽의 **휴지통 아이콘**으로 삭제할 수 있습니다.

---

#### 7. 대진표 다운로드

1. 대진표 화면 우상단 **다운로드** 버튼 클릭
2. **PNG / JPG / PDF** 중 원하는 형식을 선택하면 자동으로 저장됩니다.

---

### 게스트 (일반 관람자)

로그인 없이 아래 기능을 자유롭게 확인할 수 있습니다.

#### 홈 대시보드

앱을 열면 바로 대시보드가 표시됩니다.

- **고등부 / 중등부** 탭으로 전환하여 각 부의 대회 정보를 확인합니다.
- 같은 부에 여러 대회가 있다면 상단의 **최신 / 2차 / 3차** 버튼으로 전환할 수 있습니다.

| 서브탭 | 내용 |
|---|---|
| **현황** | 대회 진행 통계 (참가팀, 브라켓 규모, 완료 경기 수, 진행률), 라운드별 진행 현황, 최종 순위 (대회 종료 시) |
| **학교 조회** | 팀명을 검색하거나 버튼을 눌러 해당 학교의 전체 경기 일정·결과·승패 통계를 확인 |
| **대진표** | 전체 토너먼트 브라켓 시각화 |
| **결과 피드** | 완료된 경기 목록 (최신순), 스코어 및 승자 표시 |
| **공지** | 대회 관련 공지사항 확인 |

---

#### 학교별 경기 조회

1. **학교 조회** 탭 이동
2. 검색창에 학교명을 입력하거나 하단 팀 버튼을 클릭합니다.
3. 해당 팀의 모든 경기 일정, 결과, 승패 기록이 표시됩니다.

---

#### 테마 변경

화면 우하단 **테마 버튼(☀️ / 🌙 / 🖥)** 을 클릭하여 라이트 / 다크 / 시스템 자동 모드를 전환할 수 있습니다.

## 개발 환경 실행

```bash
npm install
npm run dev    # http://localhost:5173
npm run build  # 프로덕션 빌드
```

## 기술 스택

- React 19 + Vite 8
- Tailwind CSS v4
- html2canvas + jsPDF (브라켓 이미지/PDF 내보내기)
- 브라우저 `localStorage` 기반 데이터 저장 (서버 불필요)
